### Prerequisites

- Node.js 18+
- pnpm 8+
- Docker (optional, for containerized deployment)

### Installation

```bash
# Clone repository
git clone https://github.com/your-org/eigenlayer-hackathon.git
cd apps/tee

# Install dependencies
pnpm install

# Copy environment template
cp .env.example .env

# Edit .env with your API keys
# Required: WALLET_PRIVATE_KEY, GROQ_API_KEY, SERPER_API_KEY
```

### Development

```bash
# Start development server
pnpm run dev

# Server runs on http://localhost:3001
```

### Production

```bash
# Build
pnpm run build

# Start production server
NODE_ENV=production pnpm start

# Or use Docker
docker-compose up -d
```

## ğŸ”‘ Key Features

### AI-Powered Roadmaps

Generate personalized learning paths with:
- Context-aware milestones
- Live documentation references
- Progressive difficulty
- Estimated timeframes

```bash
curl -X POST http://localhost:3001/roadmap/generate-dynamic \
  -H "Content-Type: application/json" \
  -d '{"topic":"React Hooks","mode":"deep"}'
```

### Dynamic Challenges

Create coding challenges with:
- Multiple difficulty levels
- Automated grading
- Keyword-based evaluation
- Point-based scoring

### TEE Attestation

Every roadmap and challenge includes:
- Cryptographic signature
- TEE measurement verification
- On-chain attestation (optional)
- Tamper-proof records

### Web Search

AI agents can lookup:
- Latest documentation
- Current best practices
- API references
- Code examples

Powered by **Serper API** (Google Search).

### Roadmap Generation

|--------|----------|-------------|

| Method | Endpoint | Description |
|--------|----------|-------------|

### Private Keys

- Never commit `.env` files
