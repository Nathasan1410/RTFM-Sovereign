# Keyboard Shortcuts

RTFM-GPT is keyboard-first. Keep your hands on the keys.

| Key | Action | Context |
| :--- | :--- | :--- |
| `Cmd+K` | Open Command Palette | Global |
| `?` | Show Help | Global |
| `Space` | Toggle Complete | Module Detail |
| `J` | Next Module | Roadmap List |
| `K` | Previous Module | Roadmap List |
| `N` | Next Module | Module Detail |
| `P` | Previous Module | Module Detail |
| `B` | Back to Roadmap | Module Detail |
| `Esc` | Close Modal | Global |
